0xcode is a terminal-based AI coding agent that connects to 41+ LLM models through [BlockRun](https://blockrun.ai). It reads your code, edits files, runs commands, searches the web, and manages tasks ‚ÄĒ all from your terminal. You pay per request with USDC via the [x402](https://x402.org) protocol. No API keys, no accounts, no subscriptions.

## Quick Start

```bash
npm install -g @blockrun/0xcode
0xcode setup base     # Create a wallet (one-time)
0xcode                # Launch ‚ÄĒ picks model interactively
```

That's it. Fund the wallet address with USDC on Base, or use free models immediately.

### Token Optimization

Seven layers of automatic optimization keep context usage low and costs down:

1. **Thinking block stripping** ‚ÄĒ removes old reasoning from history
2. **Tool result budgeting** ‚ÄĒ caps large outputs at 50K chars with preview
3. **Microcompaction** ‚ÄĒ clears old tool results (keeps last 8)
4. **Time-based cleanup** ‚ÄĒ clears stale results after 60min idle
5. **Auto-compact** ‚ÄĒ summarizes history when approaching context limit
6. **Adaptive max_tokens** ‚ÄĒ starts at 8K, escalates to 64K on demand
7. **Prompt-too-long recovery** ‚ÄĒ auto-compacts and retries up to 3x

### Permission System

By default, read-only tools run automatically. Destructive tools (Write, Edit, Bash) prompt for permission:

```
  Permission required: Bash
  Execute: rm -rf node_modules

  Allow? [y]es / [n]o / [a]lways:
```

Use `--trust` to skip all prompts:

```bash
0xcode --trust
```

Configure rules in `~/.blockrun/0xcode-permissions.json`:

```json
{
  "allow": ["Bash(git *)"],
  "deny": ["Bash(rm -rf *)"]
}
```

### Streaming Tool Execution

Concurrent-safe tools (Read, Glob, Grep) start executing while the model is still streaming. Sequential tools (Write, Edit, Bash) wait for the full response. This reduces latency on multi-tool turns.

### Proxy Mode

0xcode also includes a proxy mode for use with Claude Code or other tools:

```bash
0xcode proxy                    # Start payment proxy on :8402
0xcode proxy -m deepseek        # With default model
0xcode init                     # Auto-configure Claude Code + LaunchAgent
```

## Architecture

```

  ‚Üí Permission check (allow / deny / ask)
  ‚Üí Agent loop
    ‚Üí Token optimization pipeline
    ‚Üí BlockRun API (direct, with x402 payment)
      ‚Üí 41+ models (Claude, GPT, Gemini, DeepSeek, Grok, ...)
    ‚Üí Streaming tool execution
    ‚Üí Context compaction (when needed)
```

## Commands

## Payment

0xcode uses the [x402](https://x402.org) protocol for pay-per-request payments with USDC stablecoins. No accounts, no API keys, no subscriptions.

**Supported chains:**
- **Base** (default) ‚ÄĒ Coinbase L2, low fees
- **Solana** ‚ÄĒ also low fees

**Setup:**

```bash
0xcode setup base     # or: 0xcode setup solana
```

Fund the wallet address with USDC. Free models work without funding.

**What does it cost?**

| Model | ~Cost per request |
|-------|-------------------|
| Free models (Nemotron, etc.) | $0 |
| DeepSeek V3 | ~$0.001 |
| Gemini Flash | ~$0.001 |
| Claude Sonnet | ~$0.01 |
| GPT-5.4 | ~$0.01 |
| Claude Opus | ~$0.05 |

Typical usage: **$5-20/month** for active development.
